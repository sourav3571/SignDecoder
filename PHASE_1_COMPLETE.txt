═══════════════════════════════════════════════════════════════════
ARCHITECTURE DECISIONS & WHY
═══════════════════════════════════════════════════════════════════

1. ASYNC/AWAIT EVERYWHERE
   ✓ Reason: FastAPI is async-native; enables high concurrency
   ✓ Benefit: Single server handles 1000+ concurrent requests
   ✓ Real-world: Mobile users with slow connections won't block others

2. SQLALCHEMY ORM
   ✓ Reason: Type-safe, queryable, relational database
   ✓ Benefit: Translation history is queryable for research
   ✓ Real-world: Can analyze "which inputs are hard to translate?"

3. REDIS FOR CACHE
   ✓ Reason: NLP models are slow; cache results
   ✓ Benefit: Same input = instant response (ms vs seconds)
   ✓ Real-world: Improve user experience with 10-100x speedup

4. MODULAR ARCHITECTURE
   ✓ Reason: Each layer is independent
   ✓ Benefit: Easy to test, replace, or scale
   ✓ Real-world: Can use different model in Phase 8 without affecting API

5. PYDANTIC VALIDATION
   ✓ Reason: Type hints + automatic validation
   ✓ Benefit: Catch bugs early; auto-generated docs
   ✓ Real-world: Never crash on bad input data

6. HEALTH CHECKS FROM DAY 1
   ✓ Reason: Production requires monitoring
   ✓ Benefit: Know immediately if something breaks
   ✓ Real-world: Users get error messages, not silent failures

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

PROBLEM: "docker-compose: command not found"
SOLUTION: Install Docker Desktop from docker.com

PROBLEM: "PostgreSQL connection failed"
SOLUTION: docker-compose restart db && sleep 5

PROBLEM: "Redis connection failed"
SOLUTION: docker-compose restart redis

PROBLEM: "Port 8000 already in use"
SOLUTION: 
  Windows: netstat -ano | findstr :8000
  Kill the process: taskkill /PID <PID> /F
  OR: Change port in run.py

PROBLEM: "ModuleNotFoundError: No module named 'app'"
SOLUTION: Make sure you're in backend/ directory when running

PROBLEM: "No such file or directory: .env"
SOLUTION: cp .env.example .env
